# 📚 BookStore API

A robust RESTful API for managing a bookstore, built with .NET 8.0 and ASP.NET Core.

## 🚀 Features

- CRUD operations for books
- Book filtering and sorting
- Fluent Validation
- Exception handling
- Flexible search capabilities

## ⚙️ Tech Stack

- .NET 8.0
- ASP.NET Core Web API
- FluentValidation
- C# 12.0
